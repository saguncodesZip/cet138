Selectors, Specificity & Cascade

Styles are applied based on selector matches; conflicts are resolved by specificity and the cascade.

/* Class vs element */
.card {{ padding: 1rem; }}
.card.highlight {{ outline: 2px solid var(--bs-primary); }}

/* Custom properties (CSS variables) */
